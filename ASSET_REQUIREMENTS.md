# Asset Requirements

The current site uses the supplied logo plus abstract generated placeholders. No Facebook media is hotlinked or publicly displayed.

## Supplied

- RaceTrack Competition logo: `public/brand/racetrack-competition-logo-source.png`
- Facebook screenshot for private reference only: `private/reference/facebook-page-reference.png`

## Temporary Digital Assets

- `public/brand/rtc-monogram.svg` is a temporary RTC monogram for favicon/mobile use, not an official redesign.
- Generated abstract `.webp` placeholders are used so pages remain visually polished without inventing fake team photographs.
- Placeholder media must not be used as proof of a specific event, podium, availability or race result.
